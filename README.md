# Database-Manager-Portfolio
🗄️ A collection of Python database projects showcasing **NoSQL** and **SQL** database management.

## 📊 Projects Included

| Project | Technology | Type | Description |
|---------|------------|------|-------------|
| **MongoDB CRUD Manager** | MongoDB Atlas + PyMongo | CLI App | Cloud NoSQL database manager with full CRUD operations |
| **FastAPI SQLite API** | FastAPI + SQLite | REST API | Web API with full CRUD and automatic Swagger documentation |
| **FastAPI MongoDB Atlas API** | FastAPI + MongoDB Atlas | REST API | Cloud NoSQL REST API with full CRUD, data validation, and Swagger UI |

---

### 🌟 Highlights

- **3 different database approaches** (CLI, local API, cloud API)
- **Both SQL and NoSQL** databases
- **Automatic API documentation** (Swagger UI)
- **Secure credential management** (.env files)
- **Full CRUD operations** on all projects
